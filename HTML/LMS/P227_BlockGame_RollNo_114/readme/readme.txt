Block Game - README

Welcome to the Flappy Bird clone game! This simple HTML5 canvas-based game is a fun project to test your reflexes.

Instructions:
1. Open the "index.html" file in a web browser to play the game.
2. Click the "ACCELERATE" button to keep the red square in the air.
3. Avoid colliding with the green obstacles to stay alive and earn points.
4. The game will keep track of your score as you progress.

Game Controls:
- Click and hold the "ACCELERATE" button to make the red square jump higher.
- Release the button to let the red square fall back down.

Objective:
Try to survive as long as possible and achieve the highest score. The game will get progressively more challenging as obstacles appear at varying heights and intervals.

Technical Details:
- The game is implemented in HTML5 and JavaScript using a simple canvas element.
- The game features a basic collision detection system and a score counter.

Customization:
Feel free to modify the game by adjusting parameters in the JavaScript code, such as obstacle frequency, speed, and more. You can also change the colors and styles as desired.

Have fun and challenge your friends to see who can achieve the highest score in this Flappy Bird-inspired game!
